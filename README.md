MonsterMod Bot
Функциональный Discord-бот для модерации и геймификации, написанный на Python (discord.py).

🛠 Возможности
Модерация
Slash-команды: Удобный ввод команд через /.

Бан/Тайм-аут: Быстрое ограничение нарушителей.

Логи: Автоматическая запись действий модераторов в специальный канал (Embed-сообщения).

Геймификация
Система опыта (XP): Пользователи получают опыт за сообщения.

Уровни: Повышение уровня при наборе определенного количества XP.

Профили: Просмотр статистики пользователя (/profile).

База данных
SQLite: Вся информация о пользователях и уровнях хранится локально в файле users.db

🚀 Установка и запуск
Клонируйте репозиторий:

Bash

git clone https://github.com/ВашНик/MonsterMod-DiscortBot.git
cd MonsterMod-DiscortBot
Установите библиотеки:

Bash

pip install -r requirements.txt
Настройка:
Создайте файл .env в корневой папке и добавьте туда токен:

Фрагмент кода

DISCORD_TOKEN=ваш_токен_тут
Запуск:

Bash

python main.py
